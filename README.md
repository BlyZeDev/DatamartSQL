# Bookstore Database – Book Exchange App

## Description

This project contains a relational database for a book exchange application. Users can lend, borrow, and rate books within a local community.

## Features

* Book management (title, ISBN, language, publisher)
* User management
* Lending system
* Ratings and comments
* Location-based data
* Delivery options

## Technologies

* MySQL
* SQL

## Setup

1. Create database:

```sql
source bookstore.sql;
```

2. Run test queries included in the script.

## Structure

* 15 tables
* Fully normalized (3NF)
* Includes test data

## Example Use Case

* User borrows a book
* Lending is stored
* User rates the book
